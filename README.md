# Bright English Studio

中小学英语成长课程网站，包含简洁首页、每日一句、预约表单、英语学习馆和独立教师管理界面。

## 英语学习馆

- 1603 个初中核心词汇，支持分层、搜索、浏览器朗读、收藏与复习
- 38 个初中语法主题，覆盖七至九年级常用语法并配有即时练习
- 学习记录使用浏览器本地存储，不要求学生注册账号

内容来源和开放数据许可见 `THIRD_PARTY_NOTICES.md`。

## 老师登录

登录成功后会跳转到 `teacher.html` 教学工作台。工作台支持：

- 课程安排、学生档案和教学待办管理
- 自动统计今日课程、在读学生和未完成待办
- JSON 数据备份与恢复
- 快速打开 Gmail 中的预约邮件

预约通过 Formspree 发送到老师邮箱，不会自动进入教师工作台。教师需要从邮箱查看预约，再手动录入工作台。

工作台数据保存在当前浏览器的 `localStorage`，不会自动跨浏览器或设备同步；导入和导出 JSON 仅用于手动备份。老师登录属于纯前端保护，不是安全的服务器端认证，不建议在工作台长期保存敏感个人信息。

预约表单使用第三方 Formspree 服务。邮件通知需要在 Formspree 中正确配置，提交仍会受到该服务的垃圾信息防护、请求频率和套餐限制；网站本身不能保证每封通知邮件都能立即送达。
