# Bright English Studio

中小学英语成长课程网站，包含简洁首页、每日一句、预约表单、英语学习馆和独立教师管理界面。

## 英语学习馆

- 1603 个初中核心词汇，支持分层、搜索、浏览器朗读、收藏与复习
- 38 个初中语法主题，覆盖七至九年级常用语法并配有即时练习
- 学习记录使用浏览器本地存储，不要求学生注册账号

内容来源和开放数据许可见 `THIRD_PARTY_NOTICES.md`。

## 老师登录

登录成功后会跳转到 `teacher.html` 教学工作台。工作台支持：

- 课程安排、学生档案和教学待办管理
- 自动统计今日课程、在读学生和未完成待办
- JSON 数据备份与恢复
- 快速打开 Gmail 中的预约邮件

工作台数据保存在当前浏览器。登录密码已移除明文并设为 8 小时会话，但 GitHub Pages 仍是静态托管，不能提供真正的服务器端权限保护。需要跨设备同步或保存敏感学生信息时，应接入云端身份认证与数据库。
