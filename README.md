## GitHub Pages 设置

如果页面还没有公开访问地址，请进入仓库：

`Settings` -> `Pages` -> `Deploy from a branch`

然后选择：

- Branch: `main`
- Folder: `/root`

保存后等待 1-3 分钟即可访问。
